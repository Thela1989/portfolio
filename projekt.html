Projekt

Rollspel

Jag har utvecklat ett bildbaserat rollspel som ett projekt för att demonstrera mina kunskaper inom webbutveckling. Spelet är byggt med hjälp av HTML för strukturen, CSS för designen och JavaScript för funktionaliteten. Det är ett interaktivt äventyrsspel där spelaren navigerar genom olika scener och gör val som påverkar spelets utgång.

Miniräknare

Jag har byggt en digital miniräknare som en del av mitt projekt för att lära mig och tillämpa webbutvecklingstekniker. Denna miniräknare är utvecklad med hjälp av HTML för struktur, CSS för design och layout, samt JavaScript för funktionalitet och interaktivitet.

Hemsida

Jag har utvecklat en komplett hemsida som en del av mitt projekt för att förbättra mina färdigheter inom webbutveckling. Hemsidan är byggd med HTML (HyperText Markup Language) och CSS (Cascading Style Sheets), vilket ger den en solid struktur och en tilltalande design.

Att-göra-lista app

Jag har utvecklat en interaktiv att-göra-lista app som ett projekt för att stärka mina färdigheter inom webbutveckling. Denna app är skapad med HTML för strukturen, CSS för designen och JavaScript för funktionaliteten, vilket gör den både användarvänlig och effektiv.
